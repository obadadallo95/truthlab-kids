# 🧪 مختبر الحقائق - منصة تعليمية لمحو الأمية الإعلامية

## 🌟 نظرة عامة

مختبر الحقائق هو منصة تعليمية تفاعلية متعددة اللغات مصممة لتمكين الشباب من مهارات التحقق من المعلومات ومحو الأمية الإعلامية في العصر الرقمي. المنصة تستخدم تقنيات متقدمة بما في ذلك الذكاء الاصطناعي والتحليلات الذكية لتوفير تجربة تعليمية شاملة وفعالة.

## 🎯 الهدف الرئيسي

- **تمكين الشباب**: تزويد الجيل القادم بمهارات التفكير النقدي والتحقق من المعلومات
- **التعلم التفاعلي**: تقديم محتوى تعليمي تفاعلي ومناسب لجميع الأعمار
- **التعددية اللغوية**: دعم أربع لغات رئيسية (العربية، الإنجليزية، الألمانية، الكورية)
- **المعايير التعليمية**: الالتزام بأعلى معايير الجودة في المحتوى التعليمي

## 🎨 معايير الترميز

### هيكل المكونات
```typescript
// مثال على مكون احترافي
interface ComponentProps {
  translations: {
    title: string;
    subtitle: string;
    // ... مفاتيح الترجمة الأخرى
  };
}

export function Component({ translations }: ComponentProps) {
  // استخدام الترجمة
  return (
    <div>
      <h1>{translations.title}</h1>
      <p>{translations.subtitle}</p>
    </div>
  );
}
```

### قواعد التسمية
- **المكونات**: PascalCase (مثال: `ProfessionalPath`)
- **الملفات**: kebab-case (مثال: `professional-path.tsx`)
- **المتغيرات**: camelCase (مثال: `userName`)
- **الثوابت**: UPPER_SNAKE_CASE (مثال: `API_BASE_URL`)

### أفضل الممارسات
1. **استخدام TypeScript** لجميع المكونات الجديدة
2. **الترجمة الكاملة** لجميع النصوص المعروضة للمستخدم
3. **الاستجابة** للشاشات المختلفة
4. **التحسين** لمحركات البحث (SEO)
5. **الاختبار** للمكونات الرئيسية

- تتبع التقدم

### 4. مختبر الحقائق (`/truth-lab`)
- أدوات التحقق من الصور
- تحقيق المصادر
- تحليل البيانات الوصفية

### 5. محاكي الانتشار (`/spread-simulator`)
- محاكاة انتشار المعلومات
- تحليل الشبكات الاجتماعية
- فهم ديناميكيات المحتوى الفيروسي

## 🔧 التطوير والصيانة

### إضافة صفحة جديدة
1. أنشئ مجلد جديد في `src/app/[locale]/`
2. أضف `page.tsx` مع المحتوى المناسب
3. أضف الترجمات في ملفات اللغات
4. حدّث التنقل إذا لزم الأمر

### إضافة مكون جديد
1. أنشئ ملف المكون في `src/components/`
2. استخدم TypeScript للتعريفات
3. أضف الترجمات اللازمة
4. اتبع معايير الترميز

### إضافة ترجمة جديدة
1. عدّل `src/i18n/routing.ts`
2. أضف ملف اللغة في `src/messages/`
3. حدّث `src/app/[locale]/layout.tsx`
4. اختبر الترجمة الجديدة
